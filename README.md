# 📊 CSV Analyzer Tool

An interactive command-line data analysis and cleaning tool built using **Python** and **Pandas**.
This tool helps users quickly inspect, clean, and prepare CSV datasets without writing complex code.

---

## 🚀 Features

✅ Load any CSV dataset

✅ View first few rows of data

✅ Get complete dataset information

✅ Generate statistical summaries

✅ Check missing values

✅ Identify duplicate rows

✅ View all column names

✅ Remove duplicate rows

✅ Fill missing values with custom input

✅ Drop rows containing missing values

✅ Rename columns easily

✅ Save cleaned dataset as a new CSV file

---


### 2️⃣ Navigate to Project Folder

```bash

```

### 3️⃣ Install Required Library

```bash
pip install pandas
```

### 4️⃣ Run the Program

```bash

```

## 🧠 How It Works

1. User enters the CSV file path
2. Tool loads dataset safely using error handling
3. Interactive menu appears with multiple analysis options
4. User selects operations to clean or analyze data
5. Cleaned dataset can be saved for further use
